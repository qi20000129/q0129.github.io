MySQL第三章总结
Posted on 2019-10-23   Updated on 2019-10-30

第三章，操作表中数据

1，SQL命令中的表达式和逻辑运算符

1.1表达式

在数据库中的表达式一般由常量，变量，函数和运算符组成。注意：单个常量，变量或函数亦可称做表达式。
SQL语言中包括三种表达反式：
第一种：<表名> 后跟<字段名表达式>
第二种：select语句后的<目标表达式>
第三种：where语句后的<条件表达式>

1.2比较运算符

比较运算符用来比较两个数的大小，其结果是1，0或者null。
运算符 含义
> 大于
>= 大于等于
< 小于
<= 小于等于
= 等于
!=或<> 不等于
is null 是否为空
is not null 是否不为空
in 判断一个值是in列表中的任意一个值
not in 判断一个值不是in列表中的任意一个值
like 通配符匹配
between and 判断一个值是否在两个值之间

1.3逻辑运算符

SQL命令支持的运算符有And(或者&&)，Or(或者||)，Not(或者!)。 逻辑运算符用来判断表达式的真假。如果表达式是真，结果返回 1。如果表达式是假，结果返回 0。
And 和Or运算符连接条件，Not否定条件。And连接条件，并且仅有当了两个条件都为真时才返回1，Or也连接两个条件，但只要任何一个为真就返回1。
当一个语句中使用了多个逻辑运算符时，运算先求Not的值，然后求And的值，最后求Or的值。

4，删除数据记录

4.1使用delete

可以根据条件删除，要被记录在日志上，大量删除速度会受影响，且表所占空间并没有被收回。

4.2使用truncate

全部删除，只留下一个表的结构定义，使用它删除的数据无法还原。

注意：truncate table命令不能用于有外键约束的表，如果表中存在外键约束，则必须使用delete命令来删除数据。
